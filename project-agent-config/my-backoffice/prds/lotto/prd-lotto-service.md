## 1. 서비스 개요

### 1.1 목적
사용자가 로또 번호를 직접 고르는 번거로움 없이, 다양한 전략으로 추천 번호를 즉시 생성할 수 있는 프론트엔드 전용 서비스.

### 1.2 기술 스택

| 구성 요소 | 기술 |
|---|---|
| Frontend | Nuxt 4 + Vue 3 + Tailwind CSS |
| 백엔드 | **없음** (순수 프론트엔드 로직) |

## 2. 사용자 스토리

### US-01: 추천 번호 생성
> **사용자로서**, 원하는 전략으로 로또 추천 번호를 즉시 생성받고 싶다.
> **그래야** 번호를 직접 고르는 수고 없이 빠르게 번호를 준비할 수 있다.

**인수 조건:**
- 전략을 선택하고 버튼 클릭 시 번호 세트가 즉시 생성된다.
- 게임 수(1~5게임)를 선택할 수 있다.
- 번호는 1~45 범위에서 중복 없이 6개가 추출된다.
- 추출된 번호는 오름차순으로 정렬되어 표시된다.
- 페이지 진입 시 기본값(완전 랜덤, 5게임)으로 자동 생성된다.

### US-02: 전략 선택
> **사용자로서**, 여러 추천 전략 중 하나를 선택하고 싶다.
> **그래야** 내가 선호하는 방식으로 번호를 뽑을 수 있다.

**인수 조건:**
- 3가지 전략(완전 랜덤 / 균형 배분 / 홀짝 균형) 중 하나를 선택할 수 있다.
- 전략을 변경해도 번호는 자동으로 재생성되지 않는다. (버튼 클릭 시에만 반영)

### US-03: 개별 게임 재생성
> **사용자로서**, 마음에 들지 않는 특정 게임만 다시 생성하고 싶다.
> **그래야** 나머지 게임은 유지하면서 원하는 번호를 얻을 수 있다.

**인수 조건:**
- 각 게임 카드에 "재생성" 버튼이 표시된다.
- 해당 버튼 클릭 시 선택된 전략으로 그 게임만 번호가 교체된다.

### US-04: 번호 복사
> **사용자로서**, 생성된 모든 번호를 클립보드에 복사하고 싶다.
> **그래야** 다른 곳에 붙여넣기 해 활용할 수 있다.

**인수 조건:**
- "전체 복사" 버튼 클릭 시 모든 게임 번호가 클립보드에 복사된다.
- 복사 완료 시 "복사됨!" 피드백이 2초간 표시된다.
- 복사 형식: `게임 1: 3, 12, 17, 28, 33, 41` (줄바꿈으로 게임 구분)

## 3. 화면 설계

### 3.1 페이지 구조

```
/lotto      -- 번호 추천 페이지
```

### 3.2 네비게이션 (사이드바)

```
Lotto 🎰
  └── 번호 추천    /lotto
```

### 3.3 페이지 UI 구성 (`/lotto`)

**헤더**
- 제목: "🎰 로또 번호 추천"
- 부제목: "전략을 선택하고 행운의 번호를 뽑아보세요."

**옵션 패널**

```
+---------------------------------------------------------------+
|  추천 전략                                                     |
|  +------------------+ +------------------+ +----------------+ |
|  | 완전 랜덤 ✓      | | 균형 배분        | | 홀짝 균형      | |
|  | 1~45에서 순수     | | 각 구간에서      | | 홀수 3개 +     | |
|  | 랜덤 6개 추출    | | 골고루 추출      | | 짝수 3개       | |
|  +------------------+ +------------------+ +----------------+ |
|                                                               |
|  게임 수                          [🎲 번호 생성]             |
|  [1] [2] [3] [4] [5]                                         |
+---------------------------------------------------------------+
```

**추천 번호 결과**

```
추천 번호                               [📋 전체 복사]

+---------------------------------------------------------------+
| GAME 1  [ 3 ] [ 12 ] [ 17 ] [ 28 ] [ 33 ] [ 41 ]   ↺ 재생성 |
+---------------------------------------------------------------+
| GAME 2  [ 5 ] [ 11 ] [ 22 ] [ 30 ] [ 38 ] [ 44 ]   ↺ 재생성 |
+---------------------------------------------------------------+
| ...                                                           |
+---------------------------------------------------------------+
```

- 번호 볼 색상: 1~10 노랑, 11~20 파랑, 21~30 빨강, 31~40 회색, 41~45 초록
- "↺ 재생성" 버튼은 카드 hover 시 표시
- 카드 등장 시 순차 페이드인 애니메이션 적용

## 4. 기능 명세

### 4.1 번호 추천 전략

| 전략 | 설명 | 로직 |
|---|---|---|
| 완전 랜덤 | 1~45 전체에서 순수 랜덤 | `shuffle(1~45).slice(0, 6)` |
| 균형 배분 | 각 구간에서 1개씩 + 나머지 1개 랜덤 | 5개 구간(1-9, 10-19, 20-29, 30-39, 40-45)에서 각 1개 추출 후 전체에서 1개 추가 |
| 홀짝 균형 | 홀수 3개 + 짝수 3개 | 홀수 pool / 짝수 pool에서 각 3개 랜덤 추출 |

### 4.2 번호 생성 UX 상세

- 게임 수 변경 후 생성 버튼 클릭 시 이전 결과를 초기화하고 새로 생성
- 전략 변경 후 생성 버튼 클릭 시 변경된 전략 적용
- 개별 재생성은 현재 선택된 전략을 그대로 사용
- 생성 중에는 버튼이 비활성화되고 로딩 표시
