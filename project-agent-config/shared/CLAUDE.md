# Workspace 공통 규칙

이 파일은 모든 프로젝트에 공통 적용되는 Claude 대화 규칙이다.
workspace 루트에 심링크로 연결되어 Claude 실행 시 자동 로드된다.

## 프로젝트 컨텍스트 로딩 규칙

유저가 특정 프로젝트 작업을 요청하면, **해당 프로젝트의 CLAUDE.md를 먼저 읽고** 작업을 시작한다.

| 프로젝트 | CLAUDE.md 경로 |
|----------|---------------|
| my-backoffice | `D:\workspace\project-my-backoffice\CLAUDE.md` |

> 새 프로젝트 추가 시 이 테이블에 경로 추가할 것.

## 대화 규칙

### 용어 교정
- 유저의 이해는 맞는데 기술 용어(빌드, 컴파일, 인터프리터 등)가 부정확할 경우, 답변 맨 마지막에 정확한 표현을 간단히 짚어준다.
- 대화 흐름을 끊지 않고, 한두 줄로 "정확한 표현: ~~" 형태로 정리.

### 응답 스타일
- 변경 요약 시 프로젝트명 명시
- 코드 변경 전 허락 받기
