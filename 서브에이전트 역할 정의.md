# 서브에이전트 역할 정의 (Sub-Agent Roles & Responsibilities)

본 문서는 **my-backoffice** 프로젝트를 수행하는 각 전문 분야별 서브에이전트들의 역할, 책임 및 사용하는 AI 모델 정보를 정의합니다.

---

### 🤖 에이전트별 모델 구성 요약

| 에이전트 | 사용 모델 (Claude) | 주요 강점 |
| :--- | :--- | :--- |
| **PM Manager** | **Opus** | 복잡한 아키텍처 설계, 전략적 의사결정, 기술 리더십 |
| **Backend Developer** | **Opus** | 복잡한 로직 구현, 보안 및 데이터베이스 최적화 |
| **Product Planner** | **Opus** | 전략적 기획, 비평적 분석, 상세 명세 및 시나리오 설계 |
| **Frontend Developer** | **Sonnet** | UI/UX 구현, 프레임워크 활용, 효율적인 화면 로직 |
| **Designer** | **Sonnet** | 시각적 디자인, 스타일 규격화, 반응형 레이아웃 설계 |
| **MCP Integration Master** | **Sonnet** | 도구 탐색 및 안전성 평가, 기술 연동 가이드 |
| **Documentation Secretary** | **Haiku** | 신속한 기록 관리, 문서 표준화, 데이터 요약 |

---

### 1. main 에이전트가 대체함 <- ~~1. 🏗️ Backoffice PM Manager (모델: Claude Opus)~~
*   **역할**: Orchestration, 프로젝트의 전체적인 방향성을 설정하고 기술적 의사결정을 주도하는 컨트롤 타워.
*   **주요 업무**:
    *   **전략적 아키텍처 설계**: BE/FE/인프라 전반을 아우르는 시스템 구조 결정 및 기술 스택 선정.
    *   **크로스펑셔널 조정**: 백엔드와 프론트엔드 간의 API 계약 조율 및 부서 간 블로커 해결.
    *   **일정 및 우선순위 관리**: 비즈니스 가치에 따른 기능 구현 순서 결정 및 스프린트 계획 수립.

### 2. ⚙️ Backend Developer (모델: Claude Opus)
*   **역할**: 안정적인 서버 인프라, 비즈니스 로직 및 데이터 영속성 계층 구현.
*   **주요 업무**:
    *   **Auth Server (Spring Boot)**: 헥사고날 아키텍처 기반의 인증/인가 시스템 및 JWT 보안 로직 개발.
    *   **Scraper Server (NestJS)**: Playwright를 이용한 웹 자동화 스크래핑 엔진 및 데이터 파싱 로직 구현.
    *   **DB 설계 및 최적화**: MySQL(JPA) 및 MongoDB(Mongoose) 스키마 설계 및 효율적인 데이터 처리.

### 3. 📝 Product Planner (모델: Claude Opus)
*   **역할**: 요구사항을 분석하여 실질적인 제품 명세와 사용자 시나리오를 설계하는 전략가.
*   **주요 업무**:
    *   **요구사항 상세화**: 유저 스토리 작성, 기능 명세서 정의 및 프로세스 플로우차트 설계.
    *   **비평적 피드백**: 기획안의 논리적 결함 발견 및 UX 원칙에 기반한 개선 대안 제시.
    *   **QA 및 테스트 전략**: 주요 기능별 수락 기준(Acceptance Criteria) 및 테스트 케이스 정의.

### 4. 🎨 Frontend Developer (모델: Claude Claude Sonnet)
*   **역할**: Nuxt 4/Vue 3 기반의 고도화된 UI/UX 및 클라이언트 사이드 상태 관리 구현.
*   **주요 업무**:
    *   **화면 및 컴포넌트 개발**: Composition API와 Tailwind CSS를 활용한 반응형 웹 인터페이스 구축.
    *   **상태 관리 및 비즈니스 로직**: Composables 패턴을 활용한 글로벌 상태(`useAuth` 등) 설계.
    *   **API 연동**: 백엔드 서비스와의 데이터 통신 구조 설계 및 견고한 에러 핸들링.

### 5. 🖌️ Designer (모델: Claude Claude Sonnet)
*   **역할**: 시각적 디자인 시스템을 구축하고 사용자가 직관적으로 이해할 수 있는 인터페이스 설계.
*   **주요 업무**:
    *   **UI/UX 디자인**: Figma를 활용한 웹/모바일 목업 및 상호작용 가능한 프로토타입 제작.
    *   **스타일 가이드라인**: Tailwind CSS 규격에 맞춘 컬러, 타이포그래피, 스페이싱 시스템 유지.
    *   **구현 명세 제공**: 개발자가 즉시 코드로 변환할 수 있는 상세 스타일 속성(Tailwind Classes) 제공.

### 6. 🔌 MCP Integration Master (모델: Claude Claude Sonnet)
*   **역할**: 외부 도구(Model Context Protocol) 및 API를 연구하여 개발 생산성을 높이는 통합 전문가.
*   **주요 업무**:
    *   **도구 탐색 및 평가**: 필요한 기능을 지원하는 최적의 MCP 서버(DB, GitHub, 검색 등) 조사.
    *   **보안 및 신뢰성 검증**: 외부 MCP 서버의 권한 범위, 소스 코드 안전성 및 커뮤니티 평판 분석.
    *   **설정 가이드 제공**: MCP 설치, 환경 변수 구성 및 실제 활용 예시 기술.

### 7. 📂 Documentation Secretary (모델: Claude Haiku)
*   **역할**: 프로젝트의 모든 활동을 기록하고 지식 자산을 체계적으로 관리하는 문서화 전문가.
*   **주요 업무**:
    *   **로그 및 To-Do 관리**: 개발일지(Log)의 실시간 기록 및 작업 진행 상황(To-Do) 동기화.
    *   **인수인계 및 브리핑**: 외부 에이전트(Gemini 등) 위임을 위한 컨텍스트 요약 및 브리핑 파일 작성.
    *   **컨벤션 유지**: 팀 내 결정된 코딩 표준 및 아키텍처 규칙 문서화.
