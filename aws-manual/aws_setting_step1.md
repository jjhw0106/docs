# AWS 초기 세팅 매뉴얼 - 1단계

## 개요

AWS 계정 생성 후 안전하게 사용하기 위한 기초 보안 및 비용 관리 설정.

---

## 1. 루트 계정 MFA 설정

> 루트 계정은 AWS 전체 제어권을 가진 계정. 유출 시 요금 폭탄 및 데이터 삭제 위험.

1. [console.aws.amazon.com](https://console.aws.amazon.com) → 루트 계정(이메일)으로 로그인
2. 우측 상단 계정명 클릭 → **보안 자격 증명**
3. **멀티 팩터 인증(MFA)** → **MFA 디바이스 할당**
4. **Authenticator app** 선택
5. 스마트폰에 Google Authenticator 또는 Authy로 QR 코드 스캔
6. 앱의 6자리 코드 두 번 입력 (30초 간격) → 완료

---

## 2. IAM 사용자 생성

> 평소 작업은 루트 대신 IAM 사용자로. 실수/유출 시 피해 범위 제한.

1. 검색창 → **IAM** → 왼쪽 메뉴 **사용자** → **사용자 생성**
2. 사용자 이름 입력 (예: `my-admin`)
3. **AWS Management Console에 대한 사용자 액세스 권한 제공** 체크
4. **IAM 사용자를 생성하고 싶음** 선택
5. 비밀번호 설정
6. 권한: **직접 정책 연결** → `AdministratorAccess` 검색 후 선택
7. 태그 스킵 → **사용자 생성**

---

## 3. IAM 로그인 URL 확인

1. IAM → **대시보드**
2. **이 계정의 IAM 사용자를 위한 로그인 URL** 복사
3. 저장해두기 (북마크 권장)
4. 로그아웃 후 해당 URL로 IAM 사용자 로그인

> 루트 로그인: 이메일 사용
> IAM 로그인: 위 URL + 사용자이름 + 비밀번호

---

## 4. IAM 사용자 결제 액세스 허용

> IAM 사용자는 기본적으로 결제 정보 접근 불가. 루트에서 허용 필요.

1. 루트 계정으로 로그인
2. 우측 상단 계정명 → **계정**
3. 스크롤 → **결제 정보에 대한 IAM 사용자 및 역할 액세스**
4. 편집 → **활성화** → 저장
5. 다시 IAM 사용자로 로그인

---

## 5. 예산 알림 설정

> 예상치 못한 과금 방지용. 임계치 초과 시 이메일 알림.

1. 검색창 → **Cost Management** (또는 결제 및 비용 관리)
2. 왼쪽 메뉴 **Budgets** → **예산 생성**
3. **월별 비용 예산** 선택
4. 예산 이름: `monthly-alert`
5. 금액: `20` (달러, 본인 기준으로 조정)
6. 이메일 입력 → **예산 생성**
